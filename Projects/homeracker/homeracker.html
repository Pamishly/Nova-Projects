HomeRacker


My 3d printed server rack.


I'm going to start this off by explainging that this isn't my own original project.HomeRacker was originally made by Kellervater. I highly recommend subscribing to their channel

You can find all the official documentation of the original project on the HomeRacker website, this page is for my custom parts and my assembly.

My rig has 3 Matx computers stacked on top of a HP Aruba 48 PoE Switch.

Operating Systems

I have my systems labeled as PC1, PC2, and PC3. PC1 being at the bottom and PC3 at the top. Their operating systems are listed below. I would also list their purpose however I use these for experimenting so it is constantly changing.

  • PC3 - Windows 10
  • PC2 - Windows 10
  • PC1 - Truenas Scale

The last piece with 2 variations is the fan mounts, the bottom mount had to be mounted differently due to its placement on the frame. The downloads can be found here

3.5in Cage

My assembly has 5 3.5in drives running on it so to mount them I modified this cage. I had run into a problem with the cage flexing out in the center area and not allowing the caddies to sit properly so I fixed it with a long nut and a bolt applying pressure horizontally across the center to keep it from flexing out. You can find them here

2.5in Mount

I also use 2.5in drives in my setup so instead of printing adapters to put them in the 3.5in cage I found a place to mount them within the rest of the rack and that can be found here

Assembly

All the custom parts and layout for this project was done in blender, I normally use Onshape but it was easiest to go blender this time and it worked. For the custom parts I had to run them through the microsoft repair program available in PrusaSlicer for them to slice properly. You can download the .blend file from this button

I printed the rack personally using an Elegoo Nepute 3 Pro with PLA+ and it has yet to give me any more problems than previously listed.
